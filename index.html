Havoc Autocomplete

Awesome Autocompleteness for healthcare apps using AngularJS

A minimal looking autocomplete directive originally developed by Angucomplete, and I added few tweaks to use it alongside Havoc.

Note: For testing results filtered for Disease or Syndrome / Neoplastic Process semantic types only.

Using Havoc Search API

<div angucomplete-alt 
  id="ex1" 
  placeholder="Search concepts" 
  pause="500" 
  selected-object="selectedConcept" 
  remote-url="http://havoc.appliedinformaticsinc.com/search/concepts" 
  remote-url-request-formatter="remoteUrlRequestFn" 
  remote-url-data-field="items" 
  title-field="term" 
  description-field="semantic_term" 
  minlength="3" input-class="form-control form-control-large" 
  match-class="highlight">
</div>
Concept: {{selectedConcept.originalObject.cui}}
